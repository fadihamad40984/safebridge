# جِسر الأمان - SafeBridge

منصة ويب لمتابعة الحالة النفسية للأطفال مع لوحة تحكم لولي الأمر

## 🎯 المميزات

- نظام مصادقة آمن (JWT + bcrypt)
- تسجيل مزاج يومي للأطفال
- لوحة تحكم تفاعلية لولي الأمر
- رسوم بيانية لتتبع الحالة النفسية
- تنبيهات ذكية عند انخفاض المزاج

## 🏗️ البنية التقنية

- **Frontend**: React.js
- **Backend**: Node.js + Express
- **Database**: MongoDB Atlas (Cloud)
- **Authentication**: JWT
- **Charts**: Recharts

## 📦 التثبيت السريع

### استخدام السكريبتات (الأسهل)
```bash
# التثبيت
.\install.ps1

# التشغيل
.\start.ps1
```

### أو يدوياً

#### Backend
```bash
cd backend
npm install
# ملف .env جاهز ومُعد مسبقاً
npm run dev
```

#### Frontend
```bash
cd frontend
npm install
npm start
```

**ملاحظة**: المشروع يستخدم MongoDB Atlas (Cloud) - لا حاجة لتثبيت MongoDB محلياً ✅

## 🚀 الاستخدام

1. **تسجيل طفل جديد**: سيتم توليد كود ولي الأمر تلقائياً
2. **تسجيل ولي الأمر**: استخدم الكود المولد للربط
3. **تسجيل المزاج اليومي**: الطفل يسجل حالته المزاجية يومياً
4. **متابعة ولي الأمر**: عرض التقارير والتنبيهات
